# My-Portfolio
Portifólio feito com HTML/CSS/JAVASCRIPT

Este é o meu portfólio pessoal desenvolvido para a disciplina de Programação Web. O projeto adota uma arquitetura **SPA (Single Page Application)**, otimizando a navegação fluida através de links âncoras e garantindo uma interface responsiva e dinâmica.

Projetos em Destaque (Foco em CyberSecurity)
O portfólio destaca ferramentas práticas focadas em segurança da informação:
1. **Scanner de Portas em Python:** Script tático para varredura e análise de portas abertas em ativos de rede.
2. **Gerador de Senhas Criptograficamente Seguro:** Algoritmo em Python baseado na biblioteca `secrets` (CSPRNG), mitigando ataques de força bruta através de aleatoriedade segura.
